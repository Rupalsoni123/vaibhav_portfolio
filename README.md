## 🚀 Getting Started

### Prerequisites
- Node.js
- npm or pnpm

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/vaibhav21soni/my_portfolio.git
   ```

2. Install dependencies
   ```bash
   cd my_portfolio
   pnpm install
   ```

3. Run the development server
   ```bash
   pnpm dev
   ```

4. Build for production
   ```bash
   pnpm build
   ```
